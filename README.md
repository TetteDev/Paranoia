# Paranoia
An updated version of the old BetterDiscord plugin DoNotTrack by Zerebos.
Supports whatever the original plugin used to with the addition additional network filtering (ie filtering requests) to prevent
calls to discords own telemetry endpoints.

There is some settings available, mostly just being able to turn on/off specific features. For most people the the default settings
are good enough.

Turning on verbose mode will bombard your dev tools with logs so dont enable it unless you really need it.
